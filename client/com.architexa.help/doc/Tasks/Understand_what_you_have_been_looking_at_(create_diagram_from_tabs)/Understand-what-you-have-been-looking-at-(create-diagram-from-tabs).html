Create diagram from tabs

After navigating in Eclipse for some time it is common for numerous tabs to accumulate. Determining which tabs are important to your current project and which have nothing to do with it can be a challenge. It is also helpful to easily create a visual representation of what you have recently been working on.

  1. To do this first right click on one of the tabs.
  2. Select Open all Java Editor Tabs As..
  3. Choose a diagram type to open.
    • Class diagrams work well for most cases.
    • Sequence diagrams are good if you have few tabs with many methods
    • Layered diagrams can work if you have many tabs, are working on refactoring, or working with code that spans many packages.

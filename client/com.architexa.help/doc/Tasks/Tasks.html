Common Tasks

To understand your code quickly, document your code efficiently, or solve other programming challenges, Architexa can help with built in support for all of the following tasks.

Understanding Tasks

  • Understand a new project, library, or section of code
  • Understand what you have been looking at (create diagram from tabs)
  • Find cycles in your architecture
  • Understand the details of a specific word or phrase by searching for it and opening the results in a diagram
  • Debug easily by using diagrams to see control flow
  • Analyze logic (and ensure all cases are handled effectively)
  • Have easier code reviews by understanding changes with diagrams

Documenting Tasks

  • Document a code concept
  • Document while you work by creating a diagram in the background
  • Quickly explain concepts to coworkers/managers
  • Keep documentation up to date

Code Review Tasks

  • Review your code changes before committing
  • Review changes committed from Team Members
  • Document the impact of recently made changes to the code architecture

Sharing Tasks

  • Share Online
  • Share via Email
  • Retrieve diagrams

Miscellaneous

  • Easily switch from one diagram type to another

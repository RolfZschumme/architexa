Document while you work

Architexa provides an easy way for you to create a diagram while exploring the code.

  1. Open the type of diagram by clicking on the "Open a new diagram editor" button (Class diagrams tend to work the best for traditional code navigation)
  2. Click the "Track Environmental Explorations" button
  3. You can also track only the tabs you've opened by clicking the "Track Open Java Editor Tabs" button instead. This will only add classes and not add the connections between methods you navigate between using F3
  4. Explore your codebase normally while leaving the editor in the background or a separate pane.
  5. Check on the diagram occasionally to see the context and relationships of the code elements you've been exploring

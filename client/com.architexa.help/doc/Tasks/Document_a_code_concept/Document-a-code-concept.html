Document a code concept

If you want to document a section of your code, take a look at the tips and best practices below:

  1. Create one or more diagrams of your concept using the best practices for Documenting Code Concepts
  2. Share your diagram with others:
    • Save your diagram locally and upload it to a source repository
    • Email you diagram to others
    • Share you diagram(s) on our secure server

Related Diagrams View

Sometimes having the context of an element or how it is used in other cases provides a much more clear understanding of a code element. Architexa provides the information about all the diagrams on the server which has the particular code unit in question in the "Related Diagram View". To access the related diagram view follow these steps:

1) Go to "Window > Show View" and select the option "Other..."

2) Select the "Related Diagrams View" in the "Architexa" folder.

3) The related view window should now be visible and when you select any code element in the current diagram the related view should show all the digrams on the server which has this code element.
